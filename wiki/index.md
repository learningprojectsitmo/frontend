### Мы используем современные инструменты для обеспечения производительности, типизации и скорости разработки.

● Core: React + Vite (Стандарт индустрии. Быстрая сборка).
● Language: TypeScript (Строгая типизация. Запрет на использование any).
● Server State: TanStack Query (Критично. Для кэширования, загрузки и синхронизации
данных с API).
● Client State: Zustand (Для локального UI-состояния: темы, модалки. Легче и проще
Redux).
● Styling: Tailwind CSS (Utility-first подход. Меньший размер CSS бандла).
● Components: shadcn/ui (Готовые, доступные компоненты на базе Radix UI + Tailwind).
● Forms: React Hook Form + Zod (Для производительности форм и валидации схем).
